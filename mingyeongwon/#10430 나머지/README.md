## 문제
<p>(A+B)%C는 ((A%C) + (B%C))%C 와 같을까?</p>

<p>(A×B)%C는 ((A%C) × (B%C))%C 와 같을까?</p>

<p>세 수 A, B, C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.</p>

## 입력
<p>첫째 줄에 A, B, C가 순서대로 주어진다. (2 ≤ A, B, C ≤ 10000)</p>

## 출력
<p>첫째 줄에 (A+B)%C, 둘째 줄에 ((A%C) + (B%C))%C, 셋째 줄에 (A×B)%C, 넷째 줄에 ((A%C) × (B%C))%C를 출력한다.</p>
